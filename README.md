# Diario de Sucesos - Angular

Diario V2 es una aplicación web sencilla y ¿moderna? desarrollada con **Angular 19** 
(standalone components) que permite a los usuarios compartir entradas diarias que **obligatoriamente comienzan con "Hoy..."**.  

Los usuarios deben dar **like** y **dislike** a otras entradas antes de poder publicar la suya propia, fomentando así alguna interacción.
